This is a geotagged image browser, sourcing from 500px and flickr. Drag the map to start images loading. You can click on an icon on the map to load that image at full size in a new tab. Links on the right side can do the same thing. Added bonus: there's also a context menu on each map marker.

Please Note: 500px geotags are often hilariously inaccurate, apparently they care more about post processing than metadata over there. If you're trying to find the location of a 500px picture be prepared to do a lot of extra homework.

Also Note: This page is using free api keys to query the maps and images. If stuff stops working it's likely because of query overruns.

Last but not least: All queries are run with the "work safe" parameter set. Sometimes some weird crap from flickr leaks through, sorry not my fault. flickr users, y'all are a bunch of freaks.
